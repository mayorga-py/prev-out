API de Predicción de Abandono Escolar

Descripción

Esta API permite predecir la probabilidad de que un estudiante abandone su carrera universitaria, basado en diversos factores psicométricos y demográficos.

Endpoints Disponibles

  • POST /upload - Subir un archivo Excel con datos de estudiantes.
  • GET /predict - Realizar una predicción basada en los datos subidos.

Ejemplo de Uso

Para subir un archivo Excel:

curl -X POST -F "file=@students_data.xlsx" http://localhost:8080/upload

Para obtener una predicción:

curl -X GET http://localhost:8080/predict
